03 一次 /api/chat

步做什麼
①使用者送出訊息（可勾選注入知識摘錄）
②關鍵字掃 SKILL / modules / cases，取 top 摘錄
③組 system：身份 + 六經速查 + 摘錄 + 免責
④Qwen 生成（enable_thinking=false）
⑤user + assistant 寫入 Postgres sessions/messages
講義沒寫的 → 應說「講義未涉及」，勿替倪師亂編。急重症先就醫。
